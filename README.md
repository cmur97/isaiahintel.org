# Isaiah Intel Institute Website

Static GitHub Pages website for Isaiah Intel Institute.

## Pages
- Home: `index.html`
- Mission: `mission.html`
- The Problem: `problem.html`
- Donate: `donate.html`
- About Us: `about.html`

## How to edit
Open the folder in VS Code, make changes, then run:

```bash
git add .
git commit -m "Update website"
git push origin main
```

## Givebutter
On `donate.html`, replace the placeholder section with your Givebutter embed code.
